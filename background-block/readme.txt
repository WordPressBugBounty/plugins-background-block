== Installation ==

### From Gutenberg Editor:
1. Go to the Block/Gutenberg Editor.
2. Search For **Section Backgrounds** block.
3. Click on the **Section Backgrounds** to add the block.

### Download & Upload:
1. Download the **Section Builder with Backgrounds** plugin (*.zip file*).
2. In your admin area, go to the **Plugins** menu and click on **Add New**.
3. Click on **Upload Plugin** and choose the **`background-block.zip`** file and click on **Install Now**.
4. Activate the plugin.

= How to apply parallax effects to my section using the Section Backgrounds block? =
From the block's right sidebar settings, find the **Scroll Animation** option. Select **parallax** and adjust the speed as desired.
